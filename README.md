# AnkiAgent

AI-powered Anki flashcard creator for language learning.

## Installation

```bash
uv sync
```

## Setup

```bash
ankiagent init
```

Requires [Anki](https://apps.ankiweb.net/) with the [AnkiConnect](https://ankiweb.net/shared/info/2055492159) add-on running.
